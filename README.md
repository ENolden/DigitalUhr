# DigitalUhr
Eine einfache digitale Uhr, umgesetzt mit HTML, CSS und JavaScript.  
Die Anwendung zeigt die aktuelle Uhrzeit in Echtzeit mit Stunden-, Minuten- und Sekundenanzeige.

## Ãœberblick
Dieses Projekt demonstriert die Verwendung des JavaScript-`Date`-Objekts zur Darstellung einer laufenden Uhr.
Der Fokus liegt auf grundlegender DOM-Manipulation und sauberem Frontend-Code.

## Funktionen
- Echtzeit-Uhr
- Anzeige von Stunden, Minuten und Sekunden
- AM/PM-Anzeige
- Responsives Layout
- Keine externen Bibliotheken oder Frameworks

## Technologien
- HTML5
- CSS3
- JavaScript
